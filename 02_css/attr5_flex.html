flex-container

  • flex 속성을 갖는 요소, flex 부모 요소라고도 함
  • 컨테이너에 적용가능 속성
    1. flex-direction : 방향
    2. flex-wrap : 요소들을 정렬하는 기능
    3. justify-content : 주축 수평 방향 정렬
    4. align-items : 주축 수직 방향 정렬
    5. align-content : 여러 행 정렬
